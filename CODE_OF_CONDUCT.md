# ARGUS-INT Code de Conduite & Éthique OSINT

## 1. Notre Engagement
Nous nous engageons à faire d'ARGUS-INT une communauté ouverte, inclusive et focalisée sur la sécurité défensive et l'investigation légitime (CTI, lutte anti-criminalité, journalisme d'investigation).

## 2. Éthique d'Utilisation
ARGUS-INT est un outil puissant. Son développement est régi par l'éthique de l'OSINT (Open Source Intelligence) :
- **Ne pas nuire** : L'outil ne doit pas être utilisé pour le doxing abusif, le harcèlement, ou l'espionnage illégal de citoyens.
- **Vérification** : Le renseignement généré par l'IA (HUMINT) doit toujours être vérifié par un humain.

## 3. Comportement dans la Communauté
- Le respect mutuel est exigé sur tous nos canaux (Matrix, GitHub).
- Le gatekeeping (mépris des débutants) n'est pas toléré.
- Le partage d'exploits zero-day fonctionnels sur nos canaux publics entraînera un bannissement immédiat.

Les mainteneurs se réservent le droit exclusif de modérer ou d'exclure tout contributeur ne respectant pas ces principes.
