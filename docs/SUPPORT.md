# ARGUS-INT Support Channels

Pour obtenir de l'aide ou coordonner des efforts d'investigation :

- **Support Technique (GitHub Issues)** : Utilisez le tracker d'issues de notre repository pour reporter des bugs logiciels clairs et reproductibles.
- **Support Communautaire (GitHub Discussions)** : Pour l'aide à l'installation, les questions d'architecture ou les retours d'expérience.
- **Canaux Sécurisés OPSEC** :
  - **Matrix Space** : E2E Encrypted. Rejoignez notre instance auto-hébergée (Voir `community/MATRIX_SETUP.md`). Idéal pour la coordination OSINT.
  - **Signal Group** : Réservé aux annonces critiques (Kill Switch, CVE Majeure).
  - **Email PGP** : Strictement réservé au programme de Bug Bounty et de disclosure de vulnérabilités (`security@argus.local`).
