# Security Policy

## Reporting a Vulnerability

Veuillez **NE PAS** ouvrir d'issue publique sur GitHub pour signaler une vulnérabilité de sécurité critique.
Suivez notre programme de Bug Bounty documenté dans :
- [Périmètre (Scope)](bugbounty/SCOPE.md)
- [Modèle de Rapport](bugbounty/DISCLOSURE_FORM.md)

Envoyez votre rapport de manière chiffrée à :
**Email** : `security@argus.local`
**Clé PGP (Fingerprint)** : `XXXX XXXX XXXX XXXX XXXX  XXXX XXXX XXXX XXXX XXXX`
*(La clé complète est disponible dans `bugbounty/PGP_KEY.asc`)*

## SLA de Réponse
- Accusé de réception : < 72 heures
- Évaluation technique : < 7 jours
- Délai d'embargo (Critical) : 14 jours (voir `docs/security/EMBARGO_POLICY.md`)
