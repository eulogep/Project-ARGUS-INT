# Rejoindre le Space Matrix ARGUS-INT

Nous utilisons un serveur Matrix fédéré et auto-hébergé pour les communications communautaires. C'est l'unique canal de support texte officiel.

## OPSEC Guidelines
1. Ne **jamais** utiliser votre adresse IP personnelle ou professionnelle. Utilisez un VPN ou le réseau Tor.
2. Créez un compte dédié à ARGUS-INT sans lier de numéro de téléphone.
3. Activez le chiffrement de bout en bout (E2E Encrypted) dès votre première connexion et sauvegardez vos clés de récupération hors-ligne.

## Comment Rejoindre
- Serveur (Homeserver) : `matrix.argus.local` (ou onion correspondant)
- Space ID : `#argus-int-space:argus.local`
- Utilisez le client **Element** (disponible sur toutes les plateformes).
